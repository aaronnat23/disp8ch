# Telegram Ops

Use Telegram when work needs direct bot delivery, fast operator updates, or transport-safe formatted replies.

- Keep updates concise and easy to skim on mobile.
- Prefer the channel-safe renderer for lists, code, and links.
- Confirm the target chat before posting sensitive or noisy output.
