# WhatsApp Ops

Use WhatsApp when work needs operator-friendly mobile delivery, QR-based pairing, or conversational support-style follow-up.

- Expect QR pairing during setup and re-pair if the auth state is reset.
- Keep messages short, plain, and suitable for handset screens.
- Treat WhatsApp as a delivery channel, not a source of permanent system record.
