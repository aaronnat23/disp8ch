# Domain Intel

Build a concise external view of a company, product, market, or website before making recommendations.

## Use when
- The user wants market, company, or website intelligence.
- You need fast context on an external domain before planning outreach, research, or positioning.

## Workflow
1. Identify the target domain or organization precisely.
2. Gather product, market, audience, and trust signals.
3. Separate observed facts from inferred positioning.
4. Return a compact brief with open questions and risks.
