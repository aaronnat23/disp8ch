# DuckDuckGo Search

Use lightweight web search when the task needs public facts quickly without a heavy research workflow.

## Use when
- The user needs a quick public-web lookup.
- You want a low-friction search path before broader multi-source research.

## Workflow
1. Use search for quick discovery, not final authority.
2. Promote primary sources once the likely target is known.
3. Keep citations or source references when the result drives a concrete action.
4. Escalate to structured web research when ambiguity stays high.
