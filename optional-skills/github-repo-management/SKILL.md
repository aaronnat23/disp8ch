# GitHub Repo Management

Handle repository-level GitHub work such as labels, milestones, repo settings, and hygiene tasks.

## Use when
- A user wants repo organization work, not just one issue or PR.
- Release preparation needs repo metadata, labels, milestones, or templates cleaned up.

## Workflow
1. Confirm the target owner and repository.
2. Make small repo-management changes in a reversible order.
3. Prefer explicit naming and consistent labels over clever structures.
4. Summarize what changed and what still needs manual review.
