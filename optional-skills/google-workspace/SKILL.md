# Google Workspace

Operate across Docs, Sheets, Drive, Gmail, and Calendar with a bias toward clean cross-tool handoffs.

## Use when
- A task spans documents, spreadsheets, inboxes, and calendar coordination.
- The user wants operational work done across Google tooling, not just one artifact.

## Workflow
1. Identify the source of truth first.
2. Keep naming and links consistent across Docs, Sheets, and Drive.
3. Prefer structured outputs the user can reuse in Workspace tools.
4. Note any auth or sharing dependencies before claiming completion.
