# SEO Content Brief

Create content briefs that balance search intent, usefulness, and editorial clarity.

## Use when
- Planning articles, landing pages, or documentation pages for discoverability.
- You need a structured content outline before writing.

## Workflow
1. Define the target audience and search intent.
2. Identify the primary topic, supporting questions, and likely comparison angles.
3. Avoid keyword stuffing; optimize for clarity and usefulness.
4. Propose an outline that answers the main task quickly, then expands.
5. Flag evidence gaps that require subject-matter review.

## Deliverable
- Search intent.
- Audience.
- Recommended outline.
- Supporting topics.
- Evidence gaps.
