# Trello

Map loose work into boards, lists, and cards using Trello-style task discipline.

## Use when
- A user wants lightweight task tracking with explicit states.
- Chat requests need to be turned into prioritized cards with owners.

## Workflow
1. Translate requests into one card per outcome, not one card per conversation.
2. Keep titles short and action-oriented.
3. Put acceptance criteria or context in the description, not the title.
4. Track blockers and owners explicitly.
