# Voice Call Playbooks

Prepare voice or call-driven workflows so spoken interactions stay short, accurate, and easy to hand off into written records.

## Use when
- The user wants call scripts, call summaries, or voice-first escalation steps.
- A voice extension or speech workflow is involved.

## Workflow
1. Define the call goal, audience, and required outcome.
2. Keep the script short, direct, and resilient to interruptions.
3. Capture facts, decisions, and follow-ups separately from conversational filler.
4. End with explicit next steps and ownership.
5. Convert the result into a written summary that can be posted back to chat or boards.

## Deliverable
- Call agenda or script.
- Key questions.
- Summary template.
- Follow-up actions and owners.
