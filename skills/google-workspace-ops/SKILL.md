# Google Workspace Ops

Operate across Google Workspace tools with reusable patterns for docs, sheets, calendar, drive, and gmail.

- Use the configured Google OAuth credentials and existing app integrations before asking for new secrets.
- For Gmail: search threads, draft replies, summarize inbox clusters, and turn actionable mail into board tasks or scheduler entries.
- For Drive: locate folders and files, check sharing state, and store canonical links in memory or documents.
- For Docs: create structured notes, meeting summaries, and working drafts; keep source links and last-updated timestamps.
- For Sheets: append rows for lightweight tracking, read tabular data for analysis, and export snapshots into documents or workflows.
- For Calendar: list upcoming events, create follow-up events, and align reminders with scheduler jobs.
- For Contacts or directory-like data: resolve owners, attendees, or stakeholders before dispatching messages or creating tasks.
- When a Google artifact becomes a project dependency, link it into hierarchy goals, data sources, or source packs instead of keeping it isolated.
