Purpose:
Inspect dynamic pages and browser-only failures when static extraction is insufficient.

Use Tools:
Navigate, snapshot, inspect console, scroll/click only when needed, and use screenshots/vision when text snapshots are insufficient.

Stop When:
The page state, error, or source content is verified.

Never:
Loop on CAPTCHA/access-denied pages or use browser turns for static pages already extracted cleanly.
