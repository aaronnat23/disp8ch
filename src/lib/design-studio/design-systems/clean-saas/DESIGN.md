# Clean SaaS

A bright, approachable marketing system.

- Light background; soft gray surfaces; rounded corners (12px).
- Single blue accent for primary CTAs; muted gray for secondary text.
- Inter for both headings and body; large, confident headings.
- Generous vertical rhythm; airy sections; gentle borders over heavy shadows.

Use `--color-*`, `--font-*`, `--space-*`, and `--radius` tokens from `tokens.css`.
