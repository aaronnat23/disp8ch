# Dashboard Ops

A neutral, information-dense system for operational tools.

- Light gray app background; white surfaces; crisp 1px borders.
- Teal accent for primary actions and active states only.
- System sans-serif; compact type scale; tight but readable spacing (6px radius).
- Favor tables, compact controls, and clear status colors. Avoid decorative heroes.

Use `--color-*`, `--font-*`, `--space-*`, and `--radius` tokens from `tokens.css`.
