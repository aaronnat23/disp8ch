When to use: a marketing landing page for a product or service.

## Required sections
- Sticky nav with product name and a primary CTA.
- Hero with a concise value proposition and supporting subhead.
- Social proof (logos or quotes) clearly labeled if illustrative.
- Feature highlights with scannable benefits.
- How it works (3-4 steps).
- Pricing tiers.
- FAQ.
- Closing CTA and footer.

## Seed HTML instructions
One standalone HTML document, CSS variables for colors/spacing/typography, and
`data-disp8ch-id` markers on every major section.

## Quality checklist
- The first viewport communicates the value proposition without scrolling.
- Copy is concise; no invented customer names, metrics, or awards.
- Buttons and links have visible hover/focus states.

## Output contract
Return only standalone HTML/CSS. Editable `data-disp8ch-id` markers required.
